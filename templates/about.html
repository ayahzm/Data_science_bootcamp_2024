About the Project

Welcome to the Al Mayadeen Web Scraper Project! This project is part of my journey as a graduate in the DGPad Data Science Bootcamp.

Project Overview

The aim of this project is to create a web scraper for the Al Mayadeen news website. This scraper collects thousands of articles, stores them in a structured database, and provides insightful visualizations and analysis through an interactive dashboard.

Key Features

  • Scrapes news articles and metadata from the Al Mayadeen website.
  • Organizes and stores articles in a MongoDB database.
  • Provides visualizations for top keywords, authors, and article statistics.
  • Includes features such as sentiment analysis and entity extraction using Natural Language Processing (NLP).
